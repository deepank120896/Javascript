Regular Expression

  1. Flags on RegEx
    1. g - global flag
    2. i - case-insensitive flag

  2. regex.EXEC(string)
    1. Returns match else null for no match
    2. without global flag "/g", exec() returns first occurence index only

  3. regex.TEST(string)
    1. Returns TRUE if regex found in string else returns FALSE

  4. string.MATCHES(regex)
    1. Matches string for regex and returns Array or Null

  5. string.SEARCH(regex)
    1. Returns index of first match or -1

  6. string.REPLACE(regex,replacement_string)
    1. Replaces occurence of regex by replacement_string
    2. Replaces all occurences if /g used with Regex
    3. If /g Not used, only first occurence is replaced

  7. META-CHARACTERS
    1. /^deep/ : '^' means regex will match if string starts with
    2. /ak$/ : '$' means regex will match if string ends with
    3. /d*k/ : '*' matches 0 or more characters
    4. /de.pank/ : '.' matches exactly one character
    5. /deepan?k/ : '?' makes character preceeded by it, optional

  8. CHARACTER SETS
    1. /^D/i : checks for start with D Case-Insesitive
